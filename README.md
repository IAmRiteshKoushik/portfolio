# Ritesh Koushik Portfolio

Personal portfolio and blog built with Astro, Tailwind CSS, and TypeScript.

## Commands

- `pnpm dev` - start the local dev server
- `pnpm build` - run Astro checks and build the site
- `pnpm preview` - preview the production build locally
- `pnpm lint` - run ESLint

## Content

- Blog posts: `src/content/blog/`
- Work entries: `src/content/work/`
- Project entries: `src/content/projects/`
- Static assets: `public/`
